# Outreach Batch — Strategy Cheat Sheet

## Sequence (over 3 weeks)

**Week 1 — warm channels first (low risk, high learning).**

| Day | Action |
|---|---|
| Mon | Post `reddit-bratislava.md` (Tue–Thu 19:00–21:00 SK is best, so post Tue evening). Reply to every comment within 2 hours. |
| Wed | Send `email-bratislava-slovakia-eu.md` to the GetData s.r.o. address (whois first). |
| Fri | Send `email-media-pitch.md` to Refresher only. Wait. Do not blast all three media at once. |

**Week 2 — English audience + first follow-ups.**

| Day | Action |
|---|---|
| Tue | Post `reddit-solotravel.md`. Different audience, no spam risk vs week 1. |
| Wed | Send `email-vertoe-com.md` (use the Stasher swap from inside the file, not literal Vertoe). |
| Fri | If GetData hasn't replied, send the 10-day follow-up. If Refresher hasn't replied, pitch Bratislavske noviny next. |

**Week 3 — long-tail + Tripadvisor.**

| Day | Action |
|---|---|
| Mon | Post `reddit-europetravel.md`. |
| Wed | Find one active Tripadvisor Bratislava thread <60 days old, post `tripadvisor-response-1.md` or `-2.md` depending on the OP's question. Do not post both on the same thread. |
| Fri | Second Tripadvisor thread if one fits. Follow-up Stasher if no reply. |

## Expected response rates (realistic)

| Channel | Send / post | Expected useful response |
|---|---|---|
| Reddit (3 posts) | 3 | 1 thread with >20 upvotes; 1 mediocre; 1 buried. Net: ~200–800 visits, 0–1 backlink. |
| Tripadvisor (2 replies) | 2 | 100% mod retention if etiquette holds; ~30–80 views per thread; small but real trickle for months. |
| bratislava-slovakia.eu | 1 email | 30–40% reply rate (small operator, direct inbox). If they say yes: tier-1 backlink for the exact ranking query. |
| Stasher / Bounce blog | 1 email | 15–25% reply rate. If yes: data partnership or guest post = tier-1 backlink. |
| Media (Refresher / BN / Startitup) | 1 at a time | 10–20% per pitch. One yes = local press article + backlink + IG/TikTok boost. |

## What success looks like

**Win condition for this batch:** **1 tier-1 backlink** (bratislava-slovakia.eu, Refresher, or Stasher) **plus 2–3 community mentions** (Reddit threads with sustained upvotes or a Tripadvisor reply that becomes the accepted answer).

**Stretch:** the media pitch lands. A Refresher article ranks for the founder's name and gives a permanent backlink + social boost. Cost: zero.

**Floor:** the Reddit posts deliver 200–500 visits over 30 days even if no backlink lands. That's still useful for direct conversions.

## Tracking

Use the sheet you already have under `marketing/03-outreach-emails.md` (the table at line 220). Add columns: `Channel`, `Sent date`, `Reply Y/N`, `Backlink Y/N`, `Notes`. Fill it the same day you send. Most outreach fails not from bad copy but from forgotten follow-ups.

## One rule

Never send two pieces from this batch in the same hour. Spread them. Bursts pattern as spam.
